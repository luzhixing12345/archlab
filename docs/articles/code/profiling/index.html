实验分析

本次实验的难度系数还是颇高的, 主要包含三部分任务量

  1. 考虑到已经涉及到指令调度和重排, 因此单周期五阶段的 ISA 已经不再适合, 需要完成一个五级流水线的 ISA 实现
  1. 需要实现对指令的重排发射
  1. 设计性能分析模块

第二部分暂时不想考虑实现一个通用型的指令调度算法(因为有点难没想好), 只考虑实现一个比较简单的依赖图优化算法
